# AC 目录

## 写作要求

- 每条 AC 编号格式：`AC-xxx-N/B/E/F-n`（N=正常，B=边界，E=异常，F=失败）
- 前置条件必须可复现（如"源存储中存在会话 S001"）
- 操作步骤必须可执行
- 预期结果必须可验证
- 只写正常流程的 AC 属于无效 AC

## 状态说明

| 状态 | 含义 |
|------|------|
| draft | 编写中 |
| proposed | 编写完成，待门禁审查 |
| active | 审查通过，当前生效 |
